http://localhost:3000/users GET - Получение всех пользователей
    200 - Успешное получение пользователей
    400 - Неверный запрос (Некорректные параметры запроса)
    500 - Ошибка сервера

http://localhost:3000/users POST - Создание нового пользователя
    201 - Пользователь успешно создан
    400 - Неверный запрос (Некорректные данные для создания пользователя)
    500 - Ошибка сервера

http://localhost:3000/users/:id GET - Получение пользователя по ID
    200 - Успешное получение пользователя
    400 - Неверный запрос (Некорректный ID пользователя)
    404 - Пользователь не найден
    500 - Ошибка сервера

http://localhost:3000/users/:id PUT - Изменение пользователя по ID
    200 - Успешное изменение пользователя
    400 - Неверный запрос (Некорректный ID пользователя или данные изменений)
    404 - Пользователь не найден
    500 - Ошибка сервера

http://localhost:3000/users/:id DELETE - Удаление пользователя по ID
    204 - Пользователь успешно удален
    400 - Неверный запрос (Некорректный ID пользователя)
    404 - Пользователь не найден
    500 - Ошибка сервера
